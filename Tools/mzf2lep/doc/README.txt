MZF2LEP by Christophe Avoinne (hlide).

This program is based on MZF2WAV and changed the original one to output a WAV and a LEP file.

You can find the executable in ../bin.

Run mzf2lep without arguments to get a brief description of its options.

There are several speed modes : fast, twice faster, thrice faster and 4 times faster. 

--- Original ----------------------------------------------------------------------------

Mzf2wav by Jeroen F. J. Laros ( available only at http://www.sharpmz.org ).

This program translates an MZF image to WAV in various ways.

You can find the executable in ../src/release/linux or in ..\src\release\w32.

Run mzf2wav without arguments to get a brief description of its options.

For more information please read mzf2wav.pdf.
